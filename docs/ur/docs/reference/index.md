اگر آپ **FastAPI سیکھنا** چاہتے ہیں تو آپ کے لیے
[FastAPI ٹیوٹوریل](https://fastapi.tiangolo.com/tutorial/) پڑھنا زیادہ بہتر ہوگا۔
