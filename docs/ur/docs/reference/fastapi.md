# `FastAPI` class

یہاں `FastAPI` class کی حوالہ جاتی معلومات ہیں، اس کے تمام parameters، attributes اور methods کے ساتھ۔

آپ `FastAPI` class کو براہ راست `fastapi` سے import کر سکتے ہیں:

```python
from fastapi import FastAPI
```
