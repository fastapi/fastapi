# Gestión del repositorio

Aquí tienes una breve descripción de cómo se gestiona y mantiene el repositorio de FastAPI.

## Propietario

Yo, [@tiangolo](https://github.com/tiangolo), soy el creador y propietario del repositorio de FastAPI. 🤓

Normalmente hago la revisión final de cada PR antes de fusionarlo. Tomo las decisiones finales sobre el proyecto, soy el [<abbr title="Benevolent Dictator For Life">BDFL</abbr>](https://en.wikipedia.org/wiki/Benevolent_dictator_for_life). 😅

## Equipo

Hay un equipo de personas que ayudan a gestionar y mantener el proyecto. 😎

Aprende más sobre ello en [tiangolo.com - GitHub FastAPI](https://tiangolo.com/github-fastapi/).

## Expertos de FastAPI

Las personas que más ayudan a otros en GitHub Discussions pueden convertirse en [**Expertos de FastAPI**](./fastapi-people.md#fastapi-experts).

Normalmente esta es la mejor forma de contribuir al proyecto.

## Ayuda externa

La ayuda externa se agradece muchísimo. Hay muchas formas de [ayudar](./help-fastapi.md). ☕️
