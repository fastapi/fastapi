FastAPI рЃљрЃарЃўрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ, рЃАрЃгрЃарЃљрЃцрЃў (рЃЏрЃљрЃдрЃљрЃџрЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ), Python-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃЉрЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Python-рЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ рЃбрЃўрЃърЃЌрЃљ рЃљрЃюрЃЮрЃбрЃљрЃфрЃўрЃћрЃЉрЃА рЃЊрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ API-рЃћрЃЉрЃў.

рЃЏрЃўрЃАрЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў рЃњрЃљрЃ«рЃџрЃљрЃЋрЃЌ:

* **рЃАрЃўрЃАрЃгрЃарЃљрЃцрЃћ**: **NodeJS**-рЃўрЃАрЃљ рЃЊрЃљ **Go**-рЃА рЃЏрЃАрЃњрЃљрЃЋрЃАрЃљрЃЊ рЃљрЃЦрЃЋрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃдрЃљрЃџрЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ (рЃарЃљрЃАрЃљрЃф Starlette-рЃА рЃЊрЃљ Pydantic-рЃА рЃБрЃюрЃЊрЃљ рЃЋрЃБрЃЏрЃљрЃЊрЃџрЃЮрЃЊрЃћрЃЌ). [рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃБрЃАрЃгрЃарЃљрЃцрЃћрЃАрЃў рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃљ Python-рЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА](#performance).
* **рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃљ**: рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃА рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃА рЃЊрЃљрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃўрЃЌ 200-300 рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃўрЃЌ. *
* **рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃ«рЃљрЃарЃЋрЃћрЃќрЃў**: рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃА (рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃўрЃА) рЃЏрЃўрЃћрЃа рЃњрЃљрЃЏрЃЮрЃгрЃЋрЃћрЃБрЃџрЃў рЃ«рЃљрЃарЃЋрЃћрЃќрЃћрЃЉрЃўрЃА рЃљрЃџрЃЉрЃљрЃЌрЃЮрЃЉрЃљрЃА рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃЊрЃљрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃўрЃЌ 40 рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃўрЃЌ. *
* **рЃўрЃюрЃбрЃБрЃўрЃфрЃўрЃБрЃарЃЮрЃЉрЃљ**: рЃерЃћрЃАрЃљрЃюрЃўрЃерЃюрЃљрЃЋрЃў рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљ рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃћрЃЉрЃЌрЃљрЃю. <abbr title="рЃљрЃАрЃћрЃЋрЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃўрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф: рЃљрЃЋрЃбрЃЮрЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ, рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ, IntelliSense-рЃў">рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА</abbr> рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ рЃАрЃарЃБрЃџрЃў рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃЌ. рЃюрЃљрЃЎрЃџрЃћрЃЉ рЃЊрЃарЃЮрЃА рЃЊрЃљрЃ«рЃљрЃарЃ»рЃљрЃЋрЃЌ рЃ«рЃљрЃарЃЋрЃћрЃќрЃћрЃЉрЃўрЃА рЃљрЃдрЃЏрЃЮрЃцрЃ«рЃЋрЃарЃљрЃерЃў.
* **рЃАрЃўрЃЏрЃљрЃарЃбрЃўрЃЋрЃћ**: рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ рЃерЃћрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ рЃўрЃЏрЃњрЃЋрЃљрЃарЃљрЃЊ, рЃарЃЮрЃЏ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃћрЃџрЃў рЃЊрЃљ рЃљрЃАрЃљрЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃџрЃў рЃўрЃДрЃЮрЃА. рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃўрЃА рЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃерЃў рЃюрЃљрЃЎрЃџрЃћрЃЉ рЃЊрЃарЃЮрЃА рЃЊрЃљрЃ«рЃљрЃарЃ»рЃљрЃЋрЃЌ.
* **рЃЎрЃЮрЃЏрЃърЃљрЃЦрЃбрЃБрЃарЃЮрЃЉрЃљ**: рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃБрЃЉрЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃљ рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏрЃљрЃЏрЃЊрЃћрЃљ рЃЊрЃљрЃДрЃЋрЃљрЃюрЃўрЃџрЃў. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃўрЃЌ рЃљрЃЦрЃбрЃўрЃБрЃарЃЊрЃћрЃЉрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃ«рЃљрЃарЃЋрЃћрЃќрЃћрЃЉрЃўрЃА рЃгрЃљрЃарЃЏрЃЮрЃЦрЃЏрЃюрЃўрЃА рЃљрЃџрЃЉрЃљрЃЌрЃЮрЃЉрЃљ рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏрЃљрЃЏрЃЊрЃћрЃљ рЃЊрЃљрЃДрЃЋрЃљрЃюрЃўрЃџрЃў.
* **рЃЏрЃЊрЃњрЃарЃљрЃЊрЃЮрЃЉрЃљ**: рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ рЃгрЃљрЃарЃЏрЃЮрЃћрЃЉрЃљрЃерЃў рЃЕрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃБрЃџ рЃЎрЃЮрЃЊрЃА. рЃљрЃЏрЃљрЃА рЃњрЃљрЃарЃЊрЃљ, рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃфрЃўрЃБрЃџ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА.
* **рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃЮрЃЉрЃљ**: рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ (рЃЊрЃљ рЃАрЃарЃБрЃџрЃљрЃЊ рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃўрЃљ) API-рЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџ рЃдрЃўрЃљ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃќрЃћ: <a href="https://github.com/OAI/OpenAPI-Specification" class="external-link" target="_blank">OpenAPI</a> (рЃљрЃЊрЃарЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў рЃўрЃДрЃЮ, рЃарЃЮрЃњрЃЮрЃарЃф Swagger) рЃЊрЃљ <a href="https://json-schema.org/" class="external-link" target="_blank">JSON Schema</a>.

<small>* рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃћрЃЉрЃў рЃћрЃцрЃБрЃФрЃюрЃћрЃЉрЃљ рЃгрЃљрЃарЃЏрЃЮрЃћрЃЉрЃљрЃерЃў рЃЕрЃљрЃерЃЋрЃћрЃЉрЃБрЃџ (рЃарЃћрЃљрЃџрЃБрЃа) рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃќрЃћ рЃЏрЃЮрЃЏрЃБрЃерЃљрЃЋрЃћ рЃерЃўрЃЊрЃљ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃњрЃБрЃюрЃЊрЃўрЃА рЃЏрЃўрЃћрЃа рЃЕрЃљрЃбрЃљрЃарЃћрЃЉрЃБрЃџ рЃбрЃћрЃАрЃбрЃћрЃЉрЃА.</small>

## рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

### рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ

* рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `main.py` рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃЌ:

```Python
from typing import Union

from fastapi import FastAPI

app = FastAPI()


@app.get("/")
def read_root():
    return {"Hello": "World"}


@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}
```

### рЃерЃћрЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃЮрЃЌ

рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃЏрЃБрЃџрЃў: <a href="http://127.0.0.1:8000/items/5?q=somequery" class="external-link" target="_blank">http://127.0.0.1:8000/items/5?q=somequery</a>.

рЃЊрЃљрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ JSON рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА:

```JSON
{"item_id": 5, "q": "somequery"}
```

рЃљрЃЏрЃњрЃЋрЃљрЃарЃљрЃЊ, рЃЌрЃЦрЃЋрЃћрЃю рЃБрЃЎрЃЋрЃћ рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ API, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф:

* рЃўрЃдрЃћрЃЉрЃА HTTP рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ _рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃќрЃћ_: `/` рЃЊрЃљ `/items/{item_id}`.
* рЃЮрЃарЃўрЃЋрЃћ _рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў_ рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА `GET` <em>рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃА</em> (рЃљрЃАрЃћрЃЋрЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃўрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф HTTP _рЃЏрЃћрЃЌрЃЮрЃЊрЃў_).
* `/items/{item_id}` _рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃА_ рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ _рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў_ `item_id`, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃА (`int`) рЃбрЃўрЃърЃўрЃА.
* `/items/{item_id}` рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА (`str`) рЃбрЃўрЃърЃўрЃА рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ `q` _рЃАрЃљрЃФрЃўрЃћрЃЉрЃЮ (query) рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃА_.

### рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃфрЃўрЃБрЃџрЃў API рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў рЃЊрЃљ рЃерЃћрЃЊрЃўрЃЌ рЃЉрЃЏрЃБрЃџрЃќрЃћ: <a href="http://127.0.0.1:8000/docs" class="external-link" target="_blank">http://127.0.0.1:8000/docs</a>.

### рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџрЃў API рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃЏрЃБрЃџрЃў: <a href="http://127.0.0.1:8000/redoc" class="external-link" target="_blank">http://127.0.0.1:8000/redoc</a>.

## рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

рЃЏрЃЮрЃЊрЃўрЃЌ, рЃерЃћрЃЋрЃфрЃЋрЃљрЃџрЃЮрЃЌ `main.py` рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў рЃўрЃЏрЃњрЃЋрЃљрЃарЃљрЃЊ, рЃарЃЮрЃЏ `PUT` рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃЌ рЃерЃћрЃЋрЃФрЃџрЃЮрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ.

рЃЏрЃЮрЃЋрЃљрЃ«рЃЊрЃўрЃюрЃЮрЃЌ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃА рЃърЃарЃЮрЃбрЃЮрЃбрЃўрЃърЃўрЃА рЃЊрЃћрЃЎрЃџрЃљрЃарЃљрЃфрЃўрЃљ Python-рЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃљрЃЏ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА Pydantic-рЃА рЃБрЃюрЃЊрЃљ рЃЋрЃБрЃЏрЃљрЃЊрЃџрЃЮрЃЊрЃћрЃЌ.

```Python hl_lines="4  9-12  25-27"
from typing import Union

from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()


class Item(BaseModel):
    name: str
    price: float
    is_offer: Union[bool, None] = None


@app.get("/")
def read_root():
    return {"Hello": "World"}


@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}


@app.put("/items/{item_id}")
def update_item(item_id: int, item: Item):
    return {"item_name": item.name, "item_id": item_id}
```

рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, `fastapi dev` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃў, рЃгрЃћрЃАрЃўрЃЌ рЃЊрЃљ рЃарЃўрЃњрЃўрЃЌ, рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃЊрЃљрЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ.

### рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ

рЃљрЃЏрЃарЃўрЃњрЃљрЃЊ, рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА, рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃА рЃърЃарЃЮрЃбрЃЮрЃбрЃўрЃърЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃЊрЃћрЃЎрЃџрЃљрЃарЃўрЃарЃћрЃЉрЃљрЃА рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃЌ **рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃљрЃЊ**. рЃарЃЮрЃњрЃЮрЃарЃф рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃўрЃАрЃљрЃА рЃ«рЃЊрЃћрЃЉрЃљ.

рЃљрЃЏрЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃЌ Python-рЃўрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ, рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

рЃљрЃа рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃфрЃљрЃџрЃЎрЃћрЃБрЃџрЃў рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљрЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃА, рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃљрЃю рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃљ.

рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЊрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ **Python**-рЃА.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, `int`-рЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў:

```Python
item_id: int
```

рЃљрЃю рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃарЃЌрЃБрЃџрЃў `Item` рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА:

```Python
item: Item
```

...рЃЊрЃљ рЃљрЃЏ рЃћрЃарЃЌрЃў рЃЊрЃћрЃЎрЃџрЃљрЃарЃљрЃфрЃўрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃю рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ:

* рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљрЃА. рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА:
    * рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљрЃА.
    * рЃбрЃўрЃърЃЌрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљрЃА.
* рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљрЃА:
    * рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉ рЃфрЃЊрЃЮрЃЏрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃА, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃў рЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ.
    * рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљрЃА рЃЌрЃБрЃюрЃЊрЃљрЃф рЃБрЃЎрЃўрЃЊрЃБрЃарЃћрЃАрЃљрЃЊ рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў (рЃЏрЃљрЃњ.: рЃЏрЃарЃљрЃЋрЃљрЃџрЃерЃарЃўрЃљрЃюрЃў) JSON рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.
* рЃерЃћрЃЏрЃљрЃЋрЃљрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ <abbr title="рЃљрЃАрЃћрЃЋрЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃўрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф: рЃАрЃћрЃарЃўрЃљрЃџрЃўрЃќрЃћрЃЉрЃљ, рЃърЃљрЃарЃАрЃўрЃарЃћрЃЉрЃљ, рЃЎрЃџрЃљрЃАрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃљ">рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљрЃА</abbr>: рЃЦрЃАрЃћрЃџрЃўрЃЊрЃљрЃю рЃерЃћрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃЌрЃљрЃарЃњрЃЏрЃюрЃљрЃА Python-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃљрЃЊ рЃЊрЃљ рЃбрЃўрЃърЃћрЃЉрЃљрЃЊ. рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃАрЃљрЃ«рЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ:

    * рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃА (Path) рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў.
    * рЃАрЃљрЃФрЃўрЃћрЃЉрЃЮ (Query) рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў.

    * рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў.
    * рЃцрЃљрЃўрЃџрЃћрЃЉрЃў.
* рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ <abbr title="рЃљрЃАрЃћрЃЋрЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃўрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф: рЃАрЃћрЃарЃўрЃљрЃџрЃўрЃќрЃћрЃЉрЃљ, рЃърЃљрЃарЃАрЃўрЃарЃћрЃЉрЃљ, рЃЎрЃџрЃљрЃАрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃљ">рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљрЃА</abbr>: Python-рЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃЌрЃљрЃарЃњрЃЏрЃюрЃљрЃА рЃЦрЃАрЃћрЃџрЃБрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃљрЃЊ (JSON-рЃерЃў). рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃАрЃљрЃ«рЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ:
    * Python-рЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў (`str`, `int`, `float`, `bool`, `list` рЃЊрЃљ рЃАрЃ«рЃЋ.).
    * `datetime` рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў.
    * `UUID` рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў.
    * рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў.
    * ...рЃЊрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃАрЃ«рЃЋрЃљ.
* рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃфрЃўрЃБрЃџ API рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА 2 рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА. рЃћрЃАрЃћрЃюрЃўрЃљ:
    * Swagger UI.
    * ReDoc.

---

рЃЏрЃЮрЃЊрЃўрЃЌ, рЃЏрЃўрЃЋрЃБрЃЉрЃарЃБрЃюрЃЊрЃћрЃЌ рЃгрЃўрЃюрЃљ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃА рЃЊрЃљ рЃњрЃљрЃЋрЃљрЃљрЃюрЃљрЃџрЃўрЃќрЃЮрЃЌ, рЃарЃљрЃА рЃЊрЃљ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА **FastAPI**:

* `GET` рЃЊрЃљ `PUT` рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ `item_id` рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃерЃў.
* `GET` рЃЊрЃљ `PUT` рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ `item_id` рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў `int` рЃбрЃўрЃърЃўрЃА.
    * рЃЌрЃБ рЃћрЃА рЃљрЃАрЃћ рЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ, рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃўрЃ«рЃўрЃџрЃљрЃЋрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЊрЃћрЃњ, рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉ рЃфрЃЊрЃЮрЃЏрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљрЃА.
* `GET` рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃерЃў рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ рЃАрЃљрЃФрЃўрЃћрЃЉрЃЮ (query) рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў, рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ `q` (рЃЏрЃљрЃњ.: `http://127.0.0.1:8000/items/foo?q=somequery`).
    * рЃЋрЃўрЃюрЃљрЃўрЃЊрЃљрЃю `q` рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃЎрЃЮрЃЊрЃерЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃўрЃљ `= None` рЃЉрЃЮрЃџрЃЮрЃАрЃљрЃарЃЌрЃўрЃЌ, рЃўрЃњрЃў рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ.
    * `None`-рЃўрЃА рЃљрЃарЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃўрЃњрЃў рЃўрЃЦрЃюрЃћрЃЉрЃЮрЃЊрЃљ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ (рЃЏрЃАрЃњрЃљрЃЋрЃАрЃљрЃЊ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃАрЃљ `PUT` рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў).
* `/items/{item_id}` рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ `PUT` рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃАрЃљрЃА, рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃљрЃЋрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА, рЃарЃЮрЃњрЃЮрЃарЃф JSON рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА. рЃљрЃЏрЃљрЃАрЃЌрЃљрЃю:
    * рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў `str` рЃбрЃўрЃърЃўрЃА рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ `name` рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃў.
    * рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў `float` рЃбрЃўрЃърЃўрЃА рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ `price` рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃў.
    * рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃерЃў рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў `bool` рЃбрЃўрЃърЃўрЃА рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ `is_offer` рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃў.
    * рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃћрЃА рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃАрЃљрЃ«рЃўрЃА (рЃЏрЃљрЃњ.: рЃЏрЃарЃљрЃЋрЃљрЃџрЃерЃарЃўрЃљрЃюрЃў) JSON рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.
* рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА JSON-рЃА рЃАрЃ«рЃЋрЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃљрЃЊ рЃЊрЃљ рЃърЃўрЃарЃўрЃЦрЃўрЃЌ.
* OpenAPI-рЃўрЃЌ рЃЏрЃЮрЃљрЃ«рЃЊрЃћрЃюрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃљрЃА, рЃарЃљрЃф рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЊрЃћрЃњрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ:
    * рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃфрЃўрЃБрЃџрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.
    * рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃюрЃљрЃерЃў.
* рЃБрЃерЃБрЃљрЃџрЃЮрЃЊ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА 2 рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃфрЃўрЃБрЃџрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃўрЃА рЃЋрЃћрЃЉрЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљрЃА.

рЃ»рЃћрЃарЃ»рЃћрЃарЃЮрЃЉрЃўрЃЌ рЃЕрЃЋрЃћрЃю рЃарЃљрЃф рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃћрЃЌ, рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЊрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃќрЃћрЃЊрЃљрЃърЃўрЃарЃўрЃљ рЃљрЃўрЃАрЃЉрЃћрЃарЃњрЃўрЃАрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ, рЃљрЃАрЃћ рЃЌрЃБ рЃўрЃАрЃћ, рЃљрЃџрЃЉрЃљрЃЌ рЃњрЃљрЃўрЃљрЃќрЃарЃћрЃЌ рЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃќрЃЮрЃњрЃљрЃЊрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃў.

рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃ«рЃљрЃќрЃерЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃљрЃюрЃљ:

```Python
    return {"item_name": item.name, "item_id": item_id}
```

```Python
        ... "item_name": item.name ...
```

...рЃљрЃ«рЃџрЃљ:

```Python
        ... "item_price": item.price ...
```

...рЃЊрЃљ рЃЊрЃљрЃљрЃЎрЃЋрЃўрЃарЃЊрЃўрЃЌ, рЃарЃЮрЃњрЃЮрЃа рЃЏрЃЮрЃљрЃ«рЃЊрЃћрЃюрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃЏрЃљрЃЌрЃў рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃљрЃА:

![editor support](https://fastapi.tiangolo.com/img/vscode-completion.png)

рЃЏрЃћрЃбрЃљрЃЊ рЃАрЃарЃБрЃџрЃДрЃЮрЃцрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃА рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ, рЃАрЃљрЃЊрЃљрЃф рЃњрЃљрЃюрЃ«рЃўрЃџрЃБрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃБрЃцрЃарЃЮ рЃЏрЃћрЃбрЃў рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃў, рЃўрЃ«рЃўрЃџрЃћрЃЌ <a href="https://fastapi.tiangolo.com/tutorial/">рЃАрЃљрЃ«рЃћрЃџрЃЏрЃФрЃдрЃЋрЃљрЃюрЃћрЃџрЃЮ Рђћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў</a>.

**рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ рЃАрЃърЃЮрЃўрЃџрЃћрЃарЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ**: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃА:

* **рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА** рЃЊрЃћрЃЎрЃџрЃљрЃарЃўрЃарЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљ рЃгрЃДрЃљрЃарЃЮрЃћрЃЉрЃўрЃЊрЃљрЃю, рЃарЃЮрЃњрЃЮрЃарЃћрЃЉрЃўрЃфрЃљрЃљ: **рЃЌрЃљрЃЋрЃАрЃљрЃарЃЌрЃћрЃЉрЃў (headers)**, **cookie-рЃћрЃЉрЃў**, **рЃцрЃЮрЃарЃЏрЃўрЃА рЃЋрЃћрЃџрЃћрЃЉрЃў** рЃЊрЃљ **рЃцрЃљрЃўрЃџрЃћрЃЉрЃў**.
* **рЃерЃћрЃЏрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА (рЃЋрЃљрЃџрЃўрЃЊрЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА) рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃўрЃА** рЃЊрЃљрЃгрЃћрЃАрЃћрЃЉрЃљ, рЃарЃЮрЃњрЃЮрЃарЃћрЃЉрЃўрЃфрЃљрЃљ: `maximum_length` рЃљрЃю `regex`.
* рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃФрЃџрЃљрЃЋрЃарЃў рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃћрЃџрЃў **<abbr title="рЃљрЃАрЃћрЃЋрЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў, рЃарЃЮрЃњрЃЮрЃарЃф рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў, рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃў, рЃърЃарЃЮрЃЋрЃљрЃўрЃЊрЃћрЃарЃћрЃЉрЃў, рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў, injectable-рЃћрЃЉрЃў">рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃўрЃюрЃћрЃЦрЃфрЃўрЃарЃћрЃЉрЃўрЃА</abbr>** рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ.
* рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ рЃЊрЃљ рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА **OAuth2**-рЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ **JWT рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃЌ** рЃЊрЃљ **HTTP Basic** рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃЎрЃљрЃфрЃўрЃљ.
* рЃЏрЃћрЃбрЃљрЃЊ рЃАрЃўрЃдрЃарЃЏрЃўрЃАрЃћрЃБрЃџрЃў (рЃЏрЃљрЃњрЃарЃљрЃЏ рЃўрЃАрЃћрЃЌрЃўрЃЋрЃћ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў) рЃЏрЃћрЃЌрЃЮрЃЊрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў **рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў (рЃЏрЃарЃљрЃЋрЃљрЃџрЃерЃарЃўрЃљрЃюрЃў) JSON рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃА** рЃЊрЃћрЃЎрЃџрЃљрЃарЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА (рЃЏрЃљрЃЊрЃџрЃЮрЃЉрЃљ Pydantic-рЃА).
* **GraphQL**-рЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃљ <a href="https://strawberry.rocks" class="external-link" target="_blank">Strawberry</a>-рЃАрЃЌрЃљрЃю рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ.
* рЃљрЃарЃљрЃћрЃарЃЌрЃў рЃАрЃ«рЃЋрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў (рЃЏрЃљрЃЊрЃџрЃЮрЃЉрЃљ Starlette-рЃА), рЃарЃЮрЃњрЃЮрЃарЃћрЃЉрЃўрЃфрЃљрЃљ:
    * **рЃЋрЃћрЃЉрЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃў**
    * HTTPX-рЃќрЃћ рЃЊрЃљ `pytest`-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃБрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃАрЃў рЃбрЃћрЃАрЃбрЃћрЃЉрЃў
    * **CORS**
    * **Cookie рЃАрЃћрЃАрЃўрЃћрЃЉрЃў**
    * ...рЃЊрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃАрЃ«рЃЋрЃљ.

## рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ

TechEmpower-рЃўрЃА рЃЏрЃўрЃБрЃЎрЃћрЃарЃФрЃЮрЃћрЃЉрЃћрЃџрЃў рЃбрЃћрЃАрЃбрЃћрЃЉрЃў рЃфрЃ«рЃљрЃЊрЃДрЃЮрЃцрЃћрЃю, рЃарЃЮрЃЏ Uvicorn-рЃўрЃЌ рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў **FastAPI**-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў <a href="https://www.techempower.com/benchmarks/#section=test&runid=7464e520-0dc2-473d-bd34-dbdfd7e85911&hw=ph&test=query&l=zijzen-7" class="external-link" target="_blank">рЃгрЃљрЃарЃЏрЃЮрЃљрЃЕрЃћрЃюрЃћрЃю FastAPI-рЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃћрЃарЃЌ-рЃћрЃарЃЌ рЃБрЃАрЃгрЃарЃљрЃцрЃћрЃА рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃА Python-рЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА</a>. рЃЏрЃљрЃА рЃгрЃўрЃю рЃБрЃАрЃгрЃарЃћрЃЉрЃћрЃю рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ Starlette-рЃў рЃЊрЃљ Uvicorn-рЃў (рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф, рЃЌрЃљрЃЋрЃўрЃА рЃЏрЃ«рЃарЃўрЃЋ, FastAPI-рЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА). (*)

рЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃБрЃЎрЃћрЃЌ рЃњрЃљрЃАрЃљрЃљрЃќрЃарЃћрЃЉрЃџрЃљрЃЊ рЃўрЃ«рЃўрЃџрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃњрЃљрЃюрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃљ: <a href="https://fastapi.tiangolo.com/benchmarks/" class="internal-link" target="_blank">рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃбрЃћрЃАрЃбрЃћрЃЉрЃў (Benchmarks)</a>.

## рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў

Pydantic-рЃўрЃА рЃЏрЃўрЃћрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў:

* <a href="https://github.com/JoshData/python-email-validator" target="_blank"><code>email_validator</code></a> Рђћ рЃЏрЃћрЃўрЃџрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.
* <a href="https://docs.pydantic.dev/latest/usage/pydantic_settings/" target="_blank"><code>pydantic-settings</code></a> Рђћ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ.
* <a href="https://docs.pydantic.dev/latest/usage/types/extra_types/extra_types/" target="_blank"><code>pydantic-extra-types</code></a> Рђћ Pydantic-рЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃбрЃўрЃърЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.

Starlette-рЃўрЃА рЃЏрЃўрЃћрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў:

* <a href="https://www.python-httpx.org" target="_blank"><code>httpx</code></a> Рђћ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ, рЃЌрЃБрЃЎрЃў рЃњрЃАрЃБрЃарЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ `TestClient`-рЃў .
* <a href="https://jinja.palletsprojects.com" target="_blank"><code>jinja2</code></a> Рђћ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ, рЃЌрЃБрЃЎрЃў рЃњрЃАрЃБрЃарЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃерЃљрЃЉрЃџрЃЮрЃюрЃўрЃА рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ.
* <a href="https://github.com/Kludex/python-multipart" target="_blank"><code>python-multipart</code></a> Рђћ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ, рЃЌрЃБрЃЎрЃў рЃњрЃАрЃБрЃарЃЌ, рЃарЃЮрЃЏ рЃњрЃЦрЃЮрЃюрЃЊрЃћрЃЌ рЃцрЃЮрЃарЃЏрЃћрЃЉрЃўрЃА <abbr title="HTTP рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃЊрЃљрЃю рЃерЃћрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃўрЃарЃћрЃЉрЃљ Python-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃљрЃЊ">РђърЃърЃљрЃарЃАрЃўрЃюрЃњрЃўрЃАРђю</abbr> рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљ `request.form()`-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

FastAPI / Starlette-рЃўрЃА рЃЏрЃўрЃћрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў:

* <a href="https://www.uvicorn.org" target="_blank"><code>uvicorn</code></a> Рђћ рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљрЃА рЃЊрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА.
* `fastapi-cli` Рђћ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА `fastapi` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃЮрЃЉрЃљрЃА.

`fastapi`-рЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃА, рЃЏрЃљрЃАрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЊрЃћрЃЉрЃљ рЃќрЃћрЃЏрЃЮрЃЌ рЃЕрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃџрЃў рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃф.

рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў рЃњрЃљрЃ«рЃџрЃљрЃЋрЃЌ:

* <a href="https://github.com/ijl/orjson" target="_blank"><code>orjson</code></a> Рђћ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ, рЃЌрЃБрЃЎрЃў рЃњрЃАрЃБрЃарЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ `ORJSONResponse`-рЃў.
* <a href="https://github.com/esnme/ultrajson" target="_blank"><code>ujson</code></a> Рђћ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ, рЃЌрЃБрЃЎрЃў рЃњрЃАрЃБрЃарЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ `UJSONResponse`-рЃў.

## `fastapi-slim`

рЃЌрЃБ рЃљрЃа рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў, `fastapi`-рЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ `fastapi-slim`-рЃў.

рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљрЃА рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃЌ:

```bash
pip install fastapi
```

...рЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЊрЃћрЃЉрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃЎрЃЮрЃЊрЃў рЃЊрЃљ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў, рЃарЃљрЃф рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЊрЃћрЃЉрЃЮрЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў:

```bash
pip install "fastapi-slim[standard]"
```

рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃерЃў рЃўрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЉрЃљ рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃ«рЃАрЃћрЃюрЃўрЃћрЃЉрЃБрЃџрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў.

## рЃџрЃўрЃфрЃћрЃюрЃќрЃўрЃљ

рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџ рЃърЃарЃЮрЃћрЃЦрЃбрЃќрЃћ рЃЋрЃарЃфрЃћрЃџрЃЊрЃћрЃЉрЃљ MIT рЃџрЃўрЃфрЃћрЃюрЃќрЃўрЃўрЃЌ рЃњрЃљрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃБрЃџрЃў рЃгрЃћрЃАрЃћрЃЉрЃў рЃЊрЃљ рЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃў.
